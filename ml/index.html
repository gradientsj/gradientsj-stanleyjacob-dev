Machine learning

Reference implementations of the algorithms behind modern machine learning, written from scratch and explained. Each page derives the math, gives a NumPy reference implementation and a PyTorch version with a hand-written backward pass, and, where the operation is GPU-bound, puts Triton and CUDA kernels beside them behind the same language switcher used elsewhere on this site, so the reference and the kernel can be read against each other line by line.

Deep learning building blocks

The submodules transformers and convnets are assembled from. These are the GPU-bound pages, so each carries Triton and CUDA implementations beside the reference.

  • Softmax

    Turns arbitrary scores into a probability distribution, and hides two ideas that matter far beyond it: the subtract-max trick that keeps exponentials finite, and the online rescaling trick that lets the reduction stream over data it never holds at once, which is the heart of FlashAttention. Implemented in NumPy, PyTorch with a custom backward, Triton, and CUDA.
