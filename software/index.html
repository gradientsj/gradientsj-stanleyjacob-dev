Engine verified against published perft counts

A desktop chess game with an engine that proves itself

Chess is rare among programs in that the hardest part has an exact answer: the number of legal move sequences from a position to a given depth is a published constant, so a move generator is either exactly right or it is not. This game's engine matches those reference counts across more than ten million positions, with castling, en passant, promotion, and every draw rule implemented. On top of it sits an AI built from negamax search with alpha-beta pruning, a quiescence search so it stops misjudging captures at the edge of its sight, and a material plus piece-square evaluation, with three difficulty levels that change search depth instead of scripting blunders. The interface is SFML: drag-and-drop pieces, legal-move dots, a promotion picker, and a board that flips when you play Black, with the search on a background thread so the window never freezes.
