Breadth-first search (BFS)

Algorithms · graphs · Jun 2026

Breadth-first search visits nodes in order of distance from the start, one level at a time, using a queue. Because it reaches nearer nodes first, it gives the shortest path in an unweighted graph.

The only structural difference from DFS is the data structure: a queue (first in, first out) instead of a stack.

When to use it

  • Shortest path or fewest steps in an unweighted graph or grid.
  • Level-order traversal of a tree.
  • Finding the nearest node that satisfies some condition.

Step by step

  1. Put the start node in a queue and record its distance as 0.
  2. Pop the front node.
  3. For each unseen neighbor, record its distance and parent, and enqueue it.
  4. Stop at the goal, then follow parents back to reconstruct the path.
from collections import deque

def bfs_shortest(graph, start, goal):
    q = deque([start])
    dist = {start: 0}
    parent = {start: None}
    while q:
        u = q.popleft()
        if u == goal:
            break
        for v in graph[u]:
            if v not in dist:
                dist[v] = dist[u] + 1
                parent[v] = u
                q.append(v)
    if goal not in dist:
        return None
    path, cur = [], goal
    while cur is not None:
        path.append(cur); cur = parent[cur]
    return path[::-1]

Complexity

O(V + E) time, O(V) space. Gives shortest paths only when every edge has the same weight; otherwise use Dijkstra.
