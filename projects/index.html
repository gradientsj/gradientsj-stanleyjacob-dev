Uptime & SLO Monitor

A self hosted service that probes a set of live public and personal endpoints on a schedule, records latency and availability in Postgres, and serves this public status page. It computes availability and latency SLIs over rolling windows, tracks error budget burn against per service objectives set in YAML, and fires multi window multi burn rate alerts so a fast burn pages immediately while a slow burn just warns. It exposes Prometheus metrics for Grafana, ships as Docker images, and comes with Terraform and a Helm chart for a Kubernetes (GKE) deploy.
